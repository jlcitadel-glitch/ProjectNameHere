Enemy Types

  • GroundPatrol — Walk, wall/ledge flip
  • Flying — No gravity, 2D flight
  • Hopping — Jump cycles, hop cooldowns
  • Stationary — Turrets, no movement

Detection Modes

  • Radius — Sphere check around enemy
  • Cone — Angle + range in facing dir
  • LineOfSight — Raycast, blocked by walls

Enemy Prefabs

  • Bat, Flying Eye
  • Goblin, Skeleton

  • Turret

Stat Formulas

StatFormula
Max HPbaseHP + (lvl-1)*hpPerLvl + job.hpBonus + STR*5
Max ManabaseMana + (lvl-1)*mpPerLvl + job.mpBonus + INT*3
Melee Mult1 + STR * 0.02
Magic Mult1 + INT * 0.02
Atk Speed1 / (1 + AGI * 0.01)
Crit Chancemin(0.5, AGI * 0.005)
Crit Damage2 + AGI * 0.01
XP to Levelfloor((L + 300 * 2^(L/7)) / 4)

Skill Data Model

  • SkillData (SO) — id, name, type, damage type, requirements, scaling per level, effects[], prefab, sound, animation
  • JobClassData (SO) — job tree, stat modifiers, growth rates, parryData, appearance, available skills
  • SkillTreeData (SO) — hierarchy of skill nodes, connections
  • SkillEffectData (SO) — individual effect template (damage/heal/buff/projectile)
  • SkillInstance — runtime state: current level, cooldown tracker

Job Stat Modifiers (per class)

  • Warrior — attackModifier, high strPerLevel, parry: ClassicParry
  • Mage — magicModifier, high intPerLevel, parry: SpellMirror
  • Rogue — attack speed + crit, high agiPerLevel, parry: ShadowStep
